# Model Configurations

**Purpose:** Model-specific settings, parameters, and selection guidance for AI inference.

**What Belongs Here:**
- Model selection criteria by use case
- Temperature and sampling parameter recommendations
- Token limit and context window specifications
- Fine-tuning guidance and parameters
- Model versioning and deprecation guidance
- Cost and performance tradeoff guidance
- Rate limiting and quota specifications

**What Does NOT Belong Here:**
- System prompts (see ../system_prompts/)
- Response patterns (see ../response_patterns/)
- Safety guidelines (see ../safety_guidelines/)
- Implementation code

**Expected Structure:**
Configuration documentation files for each model type and use case.
