# Skill Demonstrations

**Purpose:** Demonstrations of how specific skills are applied in real scenarios, showing capability and execution patterns.

**What Belongs Here:**
- Python skill demonstration (code example, best practices shown)
- TypeScript skill demonstration
- AWS skill demonstration (architecture pattern, service usage)
- Kubernetes deployment skill demonstration
- Security review skill demonstration
- Code review feedback example

**What Does NOT Belong Here:**
- Complete workflow examples (see ../workflow_executions/)
- Agent conversation examples (see ../agent_conversations/)
- Production code (see templates/code_artifacts/)
- Skill definitions (see skills/)

**Expected Structure:**
Self-contained demonstrations showing how a skill is applied with context and explanation.
