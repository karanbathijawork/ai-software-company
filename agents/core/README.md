# agents/core/

**Purpose:** Agent definitions for core leadership and governance roles. These agents drive vision, architecture, and organizational decision-making.

**Agents in This Category:**
- [AI Orchestrator](ai_orchestrator.md) — Coordinates multi-agent product delivery; manages intake, delegation, and orchestration
- [Product Manager](product_manager.md) — Translates ideas into validated, prioritized products
- [Project Manager](project_manager.md) — Ensures on-time delivery through planning, tracking, and risk management
- [Technical Lead](technical_lead.md) — Sets technical direction and maintains code quality standards
- [Solution Architect](solution_architect.md) — Designs pragmatic, scalable technical architectures

**What Does NOT Belong Here:**
- Individual contributor roles (see engineering/)
- QA and specialized roles (see quality_assurance/)
- Operational automation (see operations/)

**Expected Structure:**
One `.md` file per core agent, following the 9-section template.
