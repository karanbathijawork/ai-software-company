# Communication

**Purpose:** MCP integrations for team communication platforms, enabling agents to post updates, receive notifications, and collaborate asynchronously.

**What Belongs Here:**
- Slack integration configuration
- Microsoft Teams integration configuration
- Discord integration configuration
- Email integration configuration
- Notification and alert routing patterns
- Message formatting standards

**What Does NOT Belong Here:**
- Communication best practices (see knowledge/)
- Communication protocols or standards
- Team structure or organizational patterns
- Actual message templates (see templates/)

**Expected Structure:**
MCP configuration and integration guides for each communication platform.
