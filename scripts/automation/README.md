# Automation

**Purpose:** CI/CD and build automation scripts for continuous integration, testing, and deployment.

**What Belongs Here:**
- Build automation scripts
- Test execution automation
- Code linting and formatting automation
- Security scanning automation
- Artifact publishing scripts
- Deployment automation scripts
- Release automation scripts
- GitHub Actions, GitLab CI, or Jenkins pipeline definitions

**What Does NOT Belong Here:**
- Setup scripts (see ../setup/)
- Validation scripts (see ../validation/)
- Maintenance scripts (see ../maintenance/)
- Framework code

**Expected Structure:**
CI/CD configuration files and automation scripts organized by pipeline stage.
