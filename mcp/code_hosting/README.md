# Code Hosting

**Purpose:** MCP integrations for source control systems, enabling agents to access, analyze, and contribute to code repositories.

**What Belongs Here:**
- GitHub integration configuration and usage
- GitLab integration configuration and usage
- Gitea integration configuration and usage
- Repository access patterns and authentication
- Code search and navigation patterns
- Pull request and code review workflows

**What Does NOT Belong Here:**
- Git branching strategies (see knowledge/best_practices/)
- Code quality standards (see knowledge/best_practices/)
- Repository structure patterns (see templates/)
- Actual implementation code

**Expected Structure:**
Configuration files and integration guides for each code hosting platform supported.
