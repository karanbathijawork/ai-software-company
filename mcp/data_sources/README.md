# Data Sources

**Purpose:** MCP integrations for data access including databases, data warehouses, APIs, and document stores. Enables agents to query and retrieve information.

**What Belongs Here:**
- Database integrations (PostgreSQL, MySQL, MongoDB, etc.)
- Data warehouse integrations (Snowflake, BigQuery, Redshift)
- API integrations (REST, GraphQL)
- Document store integrations (Elasticsearch, Milvus)
- CMS integrations
- Analytics platform integrations

**What Does NOT Belong Here:**
- Data modeling or schema design (see knowledge/)
- Query optimization patterns (see knowledge/)
- Project-specific data configurations
- Implementation code

**Expected Structure:**
MCP configuration guides for each data source type and platform.
