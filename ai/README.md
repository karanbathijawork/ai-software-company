# ai/

**Purpose:** AI-specific configurations, prompts, and safety guidelines governing how the framework's AI systems reason, respond, and operate within established boundaries.

**What Belongs Here:**
- System prompts and behavioral guidelines for AI behavior
- Response formatting patterns and output structures
- Model configurations (temperature, constraints, model selection)
- Safety guidelines, content policies, and ethical boundaries
- Prompt engineering best practices and patterns

**What Does NOT Belong Here:**
- Agent definitions (see agents/)
- Implementation code or executables
- Knowledge or case studies (see knowledge/)
- Workflow orchestrations (see workflows/)
- Project-specific configurations

**Structure:**
```
ai/
├── system_prompts/     Agent role definitions and behaviors
├── response_patterns/  Output formatting and structures
├── model_configs/      Temperature, constraints, model selection
└── safety_guidelines/  Content policy, ethical boundaries
```
