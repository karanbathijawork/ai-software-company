# templates/product_documents/

**Purpose:** Templates for product-facing documents created during planning and requirements phases.

**Files in This Category:**
- [prd-template.md](prd-template.md) — Product Requirements Document template with sections for executive summary, problem, and MVP definition
- [one_input_product_prompt.md](one_input_product_prompt.md) — Prompt template for invoking AI Orchestrator with a single product idea
- [repeatable-one-input-prompt.md](repeatable-one-input-prompt.md) — Reusable prompt pattern for autonomous MVP creation from single input

**What Does NOT Belong Here:**
- Filled-in PRDs or specifications (see examples/)
- Technical specifications (see technical_specs/)
- Project plans (see project_management/)
- Implementation code

**Expected Structure:**
Template `.md` files with clear sections and placeholders for each product document type.
