# Response Patterns

**Purpose:** Output formatting patterns and structured response templates for consistent AI-generated content.

**What Belongs Here:**
- Output format templates (JSON, Markdown, structured text)
- Response structure specifications
- Section and heading conventions
- Code snippet formatting patterns
- Table and list formatting patterns
- Error response patterns
- Validation and feedback response patterns

**What Does NOT Belong Here:**
- System prompts (see ../system_prompts/)
- Model configurations (see ../model_configs/)
- Safety guidelines (see ../safety_guidelines/)
- Content templates (see templates/)

**Expected Structure:**
Pattern definition files for each response type and output format.
