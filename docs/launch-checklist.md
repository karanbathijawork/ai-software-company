# Launch Checklist

Use this checklist before shipping the first version of a product.

## Product readiness
- The core user problem is clearly addressed
- The MVP scope is clearly defined
- The main user flow works end to end

## Quality checks
- Major bugs have been tested and resolved
- Basic error handling is in place
- The product is usable on the intended device or browser

## Launch readiness
- Setup instructions are documented
- The app can be started locally or deployed successfully
- The key success metric is defined and measurable

## Communication
- The product has a short summary for users and stakeholders
- Any known limitations are clearly documented
- A next-step plan exists for post-launch iteration
