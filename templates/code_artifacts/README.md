# Code Artifacts

**Purpose:** Boilerplate code, project scaffolding, and generation templates for quickly initializing new projects and services.

**What Belongs Here:**
- Project scaffolding templates (monorepo, microservices, etc.)
- Service initialization templates
- Configuration boilerplate
- Testing setup templates
- Build and deployment boilerplate
- CI/CD pipeline templates
- Docker and container templates
- Infrastructure-as-code boilerplate

**What Does NOT Belong Here:**
- Production implementation code
- Filled-in examples (see examples/)
- Documentation templates (see product_documents/)
- Analysis or specifications (see technical_specs/)

**Expected Structure:**
Directories and template files for each project type, following language and framework conventions.
