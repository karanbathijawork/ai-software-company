# System Prompts

**Purpose:** System prompt definitions and role specifications governing AI behavior across the framework.

**What Belongs Here:**
- Agent role and persona prompts
- Behavioral guidelines and constraints
- Output format and structure specifications
- Interaction pattern definitions
- Escalation and fallback prompts
- Model-specific prompt adaptations

**What Does NOT Belong Here:**
- Response formatting patterns (see ../response_patterns/)
- Model configurations (see ../model_configs/)
- Safety guidelines (see ../safety_guidelines/)
- Agent implementations

**Expected Structure:**
Prompt definition files for each agent role and behavioral variant.
