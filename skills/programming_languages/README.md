# Programming Languages

**Purpose:** Language-specific skill definitions enabling agents to write, review, and optimize code across different programming languages.

**What Belongs Here:**
- Python skill definition (version constraints, best practices, libraries)
- TypeScript skill definition
- Go skill definition
- Java skill definition
- Rust skill definition
- C/C++ skill definition
- Any other languages used in projects

**What Does NOT Belong Here:**
- Cloud platform skills (see ../cloud_platforms/)
- DevOps tooling (see ../devops_tools/)
- Framework or library specifics (see knowledge/)
- Implementation code

**Expected Structure:**
One `.md` file per language with version, standard libraries, best practices, and common patterns.
