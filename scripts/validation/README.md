# Validation

**Purpose:** Testing and verification scripts ensuring framework functionality and configuration correctness.

**What Belongs Here:**
- Unit test suites
- Integration test suites
- Validation scripts for framework integrity
- Configuration validation scripts
- Dependency compatibility checks
- Performance benchmarking scripts
- Security scanning scripts

**What Does NOT Belong Here:**
- Setup scripts (see ../setup/)
- Maintenance scripts (see ../maintenance/)
- CI/CD automation (see ../automation/)
- Test data or fixtures (see examples/)

**Expected Structure:**
Test and validation scripts organized by test type.
