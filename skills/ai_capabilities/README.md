# AI Capabilities

**Purpose:** AI and LLM-specific expertise enabling agents to leverage language models, reasoning, and prompt engineering effectively.

**What Belongs Here:**
- Prompt engineering patterns and techniques
- LLM reasoning strategies (chain-of-thought, few-shot, etc.)
- Model selection and configuration guidance
- Fine-tuning and instruction-following approaches
- RAG (Retrieval-Augmented Generation) patterns
- Token management and cost optimization
- Model limitations and safety considerations

**What Does NOT Belong Here:**
- Programming language skills
- Cloud platform specifics
- DevOps tools
- Project-specific configurations
- Implementation code

**Expected Structure:**
Organized `.md` files covering prompt techniques, reasoning patterns, model strategies, and best practices.
