# Agent Conversations

**Purpose:** Sample transcripts demonstrating agent-to-agent interaction patterns and communication styles.

**What Belongs Here:**
- Example conversations between Product Manager and Technical Lead
- Example conversations between Frontend and Backend engineers
- Example conversations between QA and Development teams
- Example coordination conversations with DevOps
- Real or realistic conversation walkthroughs
- Multi-turn conversation examples

**What Does NOT Belong Here:**
- Complete workflow examples (see ../workflow_executions/)
- Actual project communications
- Agent definitions (see agents/)
- Communication guidelines (see knowledge/)

**Expected Structure:**
Markdown files containing formatted conversation transcripts with context and learning points.
