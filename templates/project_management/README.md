# Project Management

**Purpose:** Templates for project coordination and delivery documents used throughout the project lifecycle.

**What Belongs Here:**
- Project roadmap template
- Project plan template
- Status report template
- Risk register template
- Stakeholder communication template
- Sprint planning template
- Release notes template
- Retrospective template

**What Does NOT Belong Here:**
- Filled-in project documents (see examples/)
- Technical specifications (see technical_specs/)
- Product requirements (see product_documents/)
- Implementation code

**Expected Structure:**
Template `.md` files for each project management artifact type.
